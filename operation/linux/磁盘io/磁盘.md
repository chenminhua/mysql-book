# 磁盘

每个磁盘有一组盘片，每个盘片有两个表面。
每个表面上有一系列同心圆磁道。
每个磁道被划分为一系列扇区。

磁盘用一个连接到传动臂的读写头来读存储在磁盘表面的位。
驱动器可以将读写头定位到任意磁道上，这个操作称为寻址(seek)。

对扇区数据的访问时间主要有三个部分：寻址时间，旋转时间，传送时间。其中寻址时间和旋转时间最长。

磁盘上有一个小小的固件，称为磁盘控制器。负责记录逻辑块号和实际磁盘扇区之间的映射关系。
